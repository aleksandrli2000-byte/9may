IT-лоток (Реанимация)

[System.AntiSleep] && Cloudflare

Бот пингует себя раз в минуту. Cloudflare шлет сигнал для авторестарта при падении.

Safe.Crash(16:30)

Если сервер упал в 16:29, после перезапуска бот сначала выгрузит память в Гугл, дождется ответа, и только потом очистит телефоны.

> run diagnostic_checklist.sh

  • ERR_01: Бот молчит. Решение: Restart сервера.
  • ERR_02: Google 429. Решение: Игнор. Бот дошлет сам.
  • ERR_03: Завис 1 человек. Решение: Очистить ID из "Памяти".
